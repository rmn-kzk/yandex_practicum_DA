## Описание
Портфолио проектов выполненных в рамках курса Аналитика Данных от Яндекс Практикум

- Построение дашбордов: Tableau
- А/В-тестирование
- Работа с гипотезами
- Машинное обучение: библиотеки Scikit-learn, XGBoost
