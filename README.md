# Yandex Practicum - Аналитика Данных

## Описание
Портфолио проектов выполненных в рамках курса Аналитика Данных от Яндекс Практикум

## Основные инструменты и навыки:
- Языки: Python (Pandas, NumPy, SciPy, Statsmodels, Matplotlib, Seaborn), SQL
- Построение дашбордов: Tableau
- А/В-тестирование
- Работа с гипотезами
- Машинное обучение: библиотеки Scikit-learn, XGBoost

## Проекты:
| Название | Описание                                                    | Навыки и инструменты           |  
|-----------|-------------------|------------------------------------------------------------------|
| Исследовательский анализ данных | Исследуем архив объявлений Яндекс Недвижимость о продаже квартир в Санкт-Петербурге и соседних населённых пунктах. Задача — выполнить предобработку данных и изучить их, чтобы найти интересные особенности и зависимости, которые существуют на рынке недвижимости. | `Python`, `EDA`, `предобработка дынных` |
